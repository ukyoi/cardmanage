使用说明：

自0.5.0版后代码用C重写，尚在测试中，Bug可能较多，欢迎诸君测试Linux版。Windows平台未经调试，运行时可能出现意外的文件写入，不建议随意测试。

Linux用户：
解压到任意文件夹内，在终端中运行linux_ver文件夹中的main文件。
如：
$ <程序所在的文件夹>/linux-ver/main

Windows用户：
解压到任意文件夹内，运行Makefile编译后，在终端中运行main文件。


若要回报Bug，请到ukyoi.wordpress.com留言或尝试联系本人。
